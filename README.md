# Calculator in C<br>

## Overview<br>

This repository contains a simple calculator program written in the C programming language.<br>
The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division.<br><br>

This project is created to strengthen my understanding of C programming fundamentals including user input, conditional statements, functions, and program logic.<br><br>


## Features<br>

<ul>
<li>Addition</li>
<li>Subtraction</li>
<li>Multiplication</li>
<li>Division</li>
<li>User-friendly console interface</li>
<li>Continuous operation until user exits</li>
<li>Error handling for division by zero</li>
</ul><br>


## Technologies Used<br>

<ul>
<li>Language: C</li>
<li>Compiler: GCC</li>
<li>Platform: Windows / Linux / macOS</li>
<li>IDE: VS Code / CodeBlocks / Any C IDE</li>
</ul><br>

## How to Compile and Run<br>

### Compile<br>

<pre>
gcc calculator.c -o calculator
</pre><br>

### Run (Linux / macOS)<br>

<pre>
./calculator
</pre><br>

### Run (Windows)<br>

<pre>
calculator.exe
</pre><br>
